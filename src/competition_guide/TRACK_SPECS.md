# 算法赛道规格说明书 (Track Specifications)

本文档整理自比赛规则说明，旨在为规划 (Planning) 算法开发提供关键几何参数与逻辑流程参考。

## 1. 直线加速赛道 (Linear Acceleration)

### 1.1 赛道布置参数
| 参数项 | 数值 / 说明 |
| :--- | :--- |
| **赛道宽度** | 约 **3 m** (两侧锥桶通道宽度) |
| **起跑/计时线间距** | **0.3 m** (起跑位置到计时段起点的缓冲距离) |
| **加速区长度** | **75 m** |
| **制动区长度** | **100 m** (终点线后的停止距离) |
| **锥桶间距** | 起步区标注为 **5 m** (用于引导/约束) |
| **关键位置** | 终点线位于加速段 (75m) 末端 |

> **说明**：示意图中的“闪电形断开符号”仅表示缩略，不影响上述物理距离标定。

### 1.2 比赛流程逻辑
1.  **准备**：车辆静止于起跑线后。
2.  **触发**：起跑，行驶 0.3m 越过计时线，计时开始。
3.  **加速**：在 75m 长直道内全速行驶。
4.  **冲线**：通过终点线，计时停止。
5.  **制动**：在 100m 制动区内完全刹停。

## 2. 8字绕环测试 (Figure-8 / Skidpad)

### 2.1 几何参数
| 参数项 | 数值 / 描述 |
| :--- | :--- |
| **圆半径** | **15.25 m** |
| **圆心间距** | **18.25 m** |
| **赛道宽度** | **3.00 m** (内圆与外圆之间的行驶区域) |
| **起/终点线** | 定义为两个圆心的连线 |

### 2.2 布置细节
*   **边界标记**：
    *   左圆/右圆的**内圆内侧**布置 16 个交通锥。
    *   左圆/右圆的**外圆外侧**布置 16 个交通锥。
*   **辅助标记**：内圆之内与外圆之外会有粉笔标记（非行驶区域）。

### 2.3 跑圈流程 (测试程序)
车辆需**垂直驶入** 8 字区域，按以下顺序执行：

1.  **右圆 - 第 1 圈**：建立转向 (通常视为热身)。
2.  **右圆 - 第 2 圈**：**计时圈**。
3.  **换向**：从右圆过渡到左圆。
4.  **左圆 - 第 3 圈**：建立反向转向。
5.  **左圆 - 第 4 圈**：**计时圈**。
6.  **完赛**：完成第 4 圈后，从交叉点出口驶离赛道。

## 3. 高速避障测试 (Autocross / Obstacle Avoidance)

此项目主要考察车辆机动性、操纵性及稳定性。

### 3.1 赛道特征参数
| 参数项 | 数值 / 范围 | 说明 |
| :--- | :--- | :--- |
| **赛道最小宽度** | **3.5 m** | |
| **单圈长度** | 约 **0.805 km** | 实际视场地而定 |
| **参考均速** | 40 – 48 km/h | 算法规划速度参考 |
| **发夹弯 (Hairpin)** | 外径 ≥ 9m <br> 直径 23-45m | 极小转弯半径场景 |
| **蛇形穿桩 (Slalom)** | 桩间距 **7.62 – 12.19 m** | 直线排列锥桶 |
| **直道** | ≤ 60 m (发夹弯结尾)<br>≤ 45 m (大弯道结尾) | 限制最高车速 |

### 3.2 规则与约束
*   **传感器约束**：禁止拆除任何无人驾驶系统的传感器及驱动器。
*   **合规性**：车辆需满足 FSEC 规则。
*   **改装限制**：有人驾驶测试环节，不允许使用针对无人驾驶改装的线控程序（需回退到物理或基础电控模式，具体视执行细则）。
